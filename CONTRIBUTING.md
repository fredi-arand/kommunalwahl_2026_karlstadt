# Instructions for AI Agents and Contributors

When working on this Python project, ensure you adhere to the following strict guidelines:

1. **Virtual Environment**: Always run Python commands and scripts using the active virtual environment `.venv`.
2. **Dependencies**: Any new dependencies must be added to `requirements.txt`.
3. **Code Formatting**: Use `black` for formatting all Python code. Run `black .` before finalizing changes.
4. **Linting**: Ensure there are no linting errors by running `flake8 .`.
5. **No Warnings**: All `flake8` warnings must be resolved before proceeding (e.g., removing unused imports, fixing line lengths).
6. **Documentation**: Update the `README.md` for any architecture or setup changes.
